Age Check

Exercise: Conditionals

Instructions

When doing front end programming, you will often need to display different results on screen based on certain conditions of a data property. Use conditional statements (if, else, etc) to change the output of the program given different inputs.

Task

Design a function that will let users know what legal privileges U.S. citizens enjoy based on their age. See the conditions below. This function should take one argument (input): the age. And the function should return (output) the text string with the privileges.

  • If you are under 16, you cannot do much outside of going to school
  • If you are 16 or older, you can drive
  • If you 18 or older, you can vote
  • If you are 21 or older, you can drink alcohol
  • If you are 25 or older, you can rent a car
  • If you are 35 or older, you can run for president
  • If you are 62 or older, you collect social security benefits

Have the program print out only the most recent thing that they’ve become eligible to do, i.e. if they are 46, only print “You can run for president.”

This exercise draws the following skills

  • if/else statements
  • boolean logic
  • comparison operators
