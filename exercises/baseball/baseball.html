Baseball Lab

    Lab: Baseball Game Schedule

    Instructions

    Create two Handlebars templates. One for the date at the top, and one for each game. The second one will loop over an array using #each to create one game for each item in the games array.

    Tasks

    1. Create the Handlebars template in the HTML using a script tag. Give this one an ID of date-template. Follow the "Date markup" structure below.
    2. Write the JS to get the template code, compile it, and insert it's new HTML into the "date" div element.
    3. Next step, getting a bit harder here with the each loop. Write a second Handlebars template for each game. Give this one an ID of game-template. Follow the "Game markup" structure below.
    4. Write the JS to compile the template and insert the HTML into the "games" UL element.
